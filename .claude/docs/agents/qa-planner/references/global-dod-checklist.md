# 테스트 완료 기준 (Global DoD Checklist)

테스트 계획 "완료"를 인정하기 위한 전체 자가점검 항목이다.

## 필수 항목 (모두 통과해야 완료)

### 기능 검증
- [ ] 모든 TC가 GREEN (통과)
- [ ] 정상 케이스 (Happy Path) TC 포함
- [ ] 경계값 케이스 TC 포함
- [ ] 예외 케이스 TC 포함

### 복지AX 도메인 필수 검증
- [ ] 복지혜택 지급 중복 방지 TC 포함
- [ ] 잔액 부족 시 예외 처리 TC 포함
- [ ] 권한 없는 접근 거부 TC 포함
- [ ] 미등록/정지 가맹점 처리 TC 포함 (해당하는 경우)
- [ ] 배치 재실행 안전성 TC 포함 (배치인 경우)

### 코드 품질
- [ ] `/code-review` 실행 후 CRITICAL 0건
- [ ] 단위 테스트 커버리지: 핵심 Service 메서드 기준

### 보안
- [ ] PII 로그 노출 없음 확인
- [ ] 인증 없이 접근 가능한 엔드포인트 없음 확인
