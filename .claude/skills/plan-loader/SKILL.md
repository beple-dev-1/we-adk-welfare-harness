# plan-loader 스킬 (내부)

개발 계획서를 파싱하여 현재 세션에서 사용할 구조화된 메타 정보를 추출한다.
이 스킬은 `/develop {스코프} {과업번호}` 실행 시 자동으로 내부 호출된다.

## 입력
- 과업번호 (예: `001`)
- 계획서 경로: `target/plans/{과업번호}/{과업번호}_dev_plan.md`

## 추출 항목
- 페이즈 목록 및 각 페이즈의 대상 파일
- BE/FE 구분 (복지AX는 현재 BE 위주)
- 자동 모드 실행 조건
- 완료된 페이즈 여부 (체크박스 기준)

## 출력
- 현재 구현 대상 페이즈 요약
- 대상 파일 목록
- 남은 작업 항목
